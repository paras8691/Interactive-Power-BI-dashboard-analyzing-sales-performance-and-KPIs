# Sales Performance Dashboard — Power BI

Project Overview
This Power BI dashboard analyzes sales performance across regions, products, and time periods to identify trends and business insights.

# Objectives

- Track monthly revenue trends

- Compare regional performance

- Identify top-performing products

- Monitor KPIs like profit margin and growth rate

# Tools Used

- Power BI

- DAX

- Data Modeling

- Power Query

# Dataset
- Sample sales dataset including:

- Order Date

- Region

- Product Category

- Sales

- Profit

# Key Insights

- West region showed highest growth in Q4

- Technology category drives 45% of total profit

- Seasonal spike observed during November–December

# How to Use

1.Download the .pbix file

2.Open in Power BI Desktop

3.Interact with filters and slicers
